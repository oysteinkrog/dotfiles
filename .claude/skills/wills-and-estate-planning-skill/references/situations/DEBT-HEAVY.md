# Debt-Heavy Estate

Sometimes the main planning problem is not how to pass wealth, but how to contain disorder.

## Core Issues

- insolvent or near-insolvent estate
- personally guaranteed debt
- house, business, or vehicle with debt exceeding practical value
- dependent-support expectations that exceed actual liquidity

## Planning Moves

- focus on operational triage and realistic liquidity
- document assets, liens, and guarantees clearly
- do not let symbolic bequests obscure insolvency reality
