# Caregiver for Parent

When an adult child is doing the real work, equal-treatment defaults often stop being coherent.

## Core Issues

- authority to act
- compensation / reimbursement
- sibling fairness
- housing and transfer implications
- Medicaid lookback and elder-financial-abuse sensitivity

## Planning Moves

- written caregiver agreement if money or housing is involved
- explicit reasoning if inheritance treatment will differ
- update both generations' plans, not only the parent's
