# Situation-Specific Estate Planning

Situation files for specific life configurations that don't fit a single profession, tier, or life event.

## Files in This Directory

- **CAREGIVER-FOR-PARENT.md** — caring for aging parent, caregiver-child considerations
- **EXPATRIATE-ABROAD.md** — US citizen living abroad, foreign-domicile planning
- **IMMIGRANT-NEW-TO-US.md** — pre-immigration planning, transition to US tax residency
- **LGBTQ-PLANNING.md** — Obergefell, equal rights, continuing challenges
- **SAME-SEX-MARRIED-COUPLE.md** — federal recognition, some state edge cases
- **POLYAMOROUS-OR-UNCONVENTIONAL-FAMILY.md** — non-traditional family structures
- **RELIGIOUS-MINORITY.md** — specific religious requirements in planning
- **GRANDPARENT-RAISING-GRANDCHILD.md** — kinship care, adoption, SSI considerations
- **ESTRANGED-FAMILY.md** — disinheritance, no-contact provisions, next-of-kin issues
- **FAMILY-BUSINESS-OWNER.md** — succession across generations
- **DEBT-HEAVY.md** — high-debt individuals, estate with more liabilities than assets
- **RECOVERY-FROM-ADDICTION.md** — beneficiary with addiction history, incentive trusts
- **AGING-ALONE.md** — single, no close family, strangers as executor

These are not decorative overlays. If one is relevant, it should appear in
`analyses/plan-coverage-matrix.md` and usually trigger either conflict-prevention,
litigation-defense, or fiduciary-scoring work.
