# Software Engineer

Tech workers often concentrate wealth in employer equity and accumulate meaningful digital assets.

## Load This File When

- the user has RSUs, options, ESPP, startup equity, or substantial digital assets

## Special Planning Issues

- concentrated employer stock
- startup equity and pre-IPO planning
- remote-work multi-state domicile drift
- digital infrastructure and security continuity

## Documents To Collect

- equity-plan statements and grant docs
- brokerage and 401(k) beneficiary records
- domain / IP / side-project ownership records
- digital-vault / password-manager setup

## Common Mistakes

- treating employer equity as "just another brokerage asset"
- failing to document digital access for a technically sophisticated but single-threaded setup
