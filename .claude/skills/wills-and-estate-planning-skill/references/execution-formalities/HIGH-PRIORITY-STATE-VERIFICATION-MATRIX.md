Use it when a plan touches one of these states and execution / formality details could
determine whether the plan works.

Log every live verification in `analyses/official-source-log.md`.

## Use Pattern

For each state above:

1. Read the controlling statute page.
2. Note the exact issue being verified.
3. Record the practical takeaway in `analyses/official-source-log.md`.
4. If the issue affects signing logistics, add it to `deliverables/attorney-engagement-brief.md`.

---

## Anti-Pattern

Do not collapse these states into a single national signing recipe.

The user may need:

- different witness arrangements,
- different self-proving mechanics,
- different deed strategy,
- or a full redraft after moving.
