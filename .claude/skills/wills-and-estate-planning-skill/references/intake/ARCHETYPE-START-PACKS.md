# Archetype Start Packs

Use these when the user wants the skill to "just start in the right place" rather than
walking from zero through a generic intake.

Each start pack gives:

- first questions
- likely first references
- likely first documents to request
- likely early outputs

Do not let archetypes replace real intake. They are accelerators, not substitutes.

---

## 1. Young Parents With House + 401(k)

Use when the user is married or partnered, has minor children, and most wealth is the
home + retirement + life insurance.

- First questions: guardians, backups, life insurance sufficiency, beneficiary forms,
  house titling, 529 continuity / successor-owner setup
- Likely first references: `MINOR-CHILDREN.md`, `RETIREMENT-ACCOUNTS.md`,
  `PRIMARY-RESIDENCE.md`, tier 1 or 2
- Likely first documents: beneficiary confirmations, deed, mortgage, insurance policies
- Likely early outputs: beneficiary map, guardian choices, implementation ledger

## 2. Second Marriage / Blended Family

- First questions: what happens if survivor remarries, who ultimately gets what,
  separate vs marital assets, ex-spouse cleanup
- Likely first references: `BLENDED-FAMILY.md`, `DIVORCED-OR-SEPARATED.md`,
  `CREDIT-SHELTER-QTIP.md` if wealth justifies
- Likely early outputs: conflict-prevention plan, litigation-risk memo, fiduciary scorecard

## 3. Aging Widow / Widower

- First questions: current helpers, cognitive decline concerns, healthcare agents,
  loneliness / executor practicalities, stale beneficiaries
- Likely first references: `AGING-ALONE.md`, incapacity references, maintenance OS
- Likely early outputs: document-quality triage, fiduciary bench, signing readiness if age/health risk

## 4. Founder / Executive / Concentrated Stock Holder

- First questions: equity comp, liquidity timeline, pre-IPO or post-IPO, concentrated
  position, DSUE / gifting history
- Likely first references: profession overlay, `INVESTMENT-ACCOUNTS-MARGIN.md`,
  `STEP-UP-BASIS-PLANNING.md`, tier 3-5
- Likely early outputs: tax analysis, confidence register, institutional operations review

## 5. Crypto-Heavy / Digital-Native User

- First questions: self-custody vs exchange, seed-phrase storage, dead-man-switch
  logic, online business / domains, legacy contacts
- Likely first references: `CRYPTO-AND-DIGITAL.md`, `DIGITAL-LEGACY.md`
- Likely early outputs: digital inventory, if-I-die-tomorrow packet, implementation ops pack

## 6. Family-Business Owner / Industrialist

- First questions: who can run payroll tomorrow, ownership structure, voting vs
  economics, buy-sell, partner agreements, family employment politics
- Likely first references: `PRIVATE-BUSINESS.md`, `BUSINESS-CONTINUITY.md`,
  `FAMILY-BUSINESS-OWNER.md`, tier 3-5
- Likely early outputs: business continuity activation, tax/liquidity analysis, litigation memo
