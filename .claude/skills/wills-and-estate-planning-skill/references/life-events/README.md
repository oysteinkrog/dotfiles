# Life Event Triggers for Estate Plan Review

Major life events require estate plan review. Some require only minor beneficiary updates; others trigger comprehensive re-planning.

## Files in This Directory

- **MARRIAGE.md** — new spouse implications, prenup, community property election, beneficiary updates
- **DIVORCE.md** — removing ex-spouse, updating beneficiaries (beyond will), child support/maintenance intersection
- **CHILD-BIRTH-ADOPTION.md** — guardian nomination, beneficiary designation, 529 planning
- **MOVING-STATES.md** — domicile establishment, document review under new state law
- **INHERITANCE-RECEIVED.md** — structuring, separate-property maintenance, tax planning
- **BUSINESS-SALE-OR-IPO.md** — pre- and post-liquidity planning, exemption use
- **RETIREMENT.md** — plan review, beneficiary update, LTC evaluation
- **DIAGNOSIS-SERIOUS-ILLNESS.md** — POLST, healthcare plan activation, rapid planning
- **BECOMING-CAREGIVER.md** — caregiver agreement, own planning review
- **DEATH-OF-FAMILY-MEMBER.md** — executor duties, personal plan review
- **STARTING-BUSINESS.md** — entity structure, buy-sell planning, succession
- **NEW-PROPERTY-PURCHASE.md** — titling, trust funding, state implications

Each file covers:
- What triggers it
- What to review
- New documents needed
- Beneficiary / title changes
- Timing considerations
- Common mistakes at this transition

Life-event reviews should usually run in `life-event-delta` or `maintenance-review`
mode rather than reflexively triggering a full greenfield rebuild.
