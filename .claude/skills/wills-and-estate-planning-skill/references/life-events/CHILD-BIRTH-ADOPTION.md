# Child Birth or Adoption

Adding a child is one of the cleanest triggers for immediate estate-plan revision.

## Immediate Priorities

- Nominate primary and backup guardians
- Create or update trusts for minors
- Update life-insurance beneficiaries and coverage amounts
- Update retirement-account beneficiaries
- Add temporary caregiver / emergency authorization planning if parents travel or have irregular schedules

## Questions To Ask

- Who would raise the child if both parents died?
- Would the same person manage the money?
- At what ages should the child receive control over inherited assets?
- Are there stepchildren, donor-conception, surrogacy, or adoption-finalization issues?
- Are there grandparents or others expected to contribute or control education funds?

## Planning Adaptations

- Use a trust, not an outright gift, for any minor inheritance
- Separate guardianship decisions from trustee decisions if different skill sets are needed
- Coordinate 529 successor-owner continuity and UTMA / UGMA accounts
- Re-check employer benefits, especially basic life insurance

## Documents To Collect

- Birth certificate or adoption papers when available
- Existing beneficiary confirmations
- 529 statements
- Life-insurance policies
- Prior wills / trusts

## Common Mistakes

- Naming minors directly as beneficiaries
- Naming one grandparent as guardian with no backup
- Forgetting beneficiary forms because "the will handles it"
- Treating college funding and inheritance control as the same question
