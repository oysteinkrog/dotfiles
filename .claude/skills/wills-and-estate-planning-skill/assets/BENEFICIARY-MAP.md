## Coherence Audit

- [ ] Every retirement account has primary AND contingent beneficiary
- [ ] No ex-spouse on any account
- [ ] No "estate" as beneficiary unless intentional
- [ ] No minor child as direct beneficiary (use trust)
- [ ] Real estate titling matches intent
- [ ] All accounts have current beneficiary form on file (verified in writing)
- [ ] State of execution matches state of domicile
- [ ] All updates documented with date

## Review Triggers

Update this map immediately upon:
- Marriage / divorce / separation
- Birth / adoption
- Death of any named beneficiary
- Major change in relationship with named beneficiary
- Move to new state
- Acquisition of new account or asset
- Change in tax law affecting designations
