# Scenario Pack: Pro Recovery, File-Heavy

Use this scenario when:
- official export scope is too weak for the target
- files matter more than perfect whole-org completeness
- the operator must rely on `slackdump` visibility plus strong gap reporting

## Required Additions

- explicit blind-spot ledger
- aggressive attachment validation
- manual rebuild backlog for integrations/workflows
- stronger stakeholder expectation-setting before cutover
