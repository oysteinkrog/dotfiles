# Phase 1 Operator Router

Use this to route the first action by persona.

## Slack Admin

First action: confirm plan tier, export approval, and whether official export can be the source of truth.

## Migration Lead

First action: pick the branch in `references/specs/CROSS-PHASE-STATE-MACHINE.md` and assign owners for export, validation, and handoff.

## Security / Compliance Reviewer

First action: run the legal/compliance gate and define retention, notification, and evidence requirements before export starts.

## AI Agent Under Pressure

First action: open `references/START-HERE.md`, determine authoritative source, and stop if plan tier or approval is unknown.

## Infra Admin Waiting On Phase 1

First action: demand `handoff.json`, manifests, counts, and explicit gap dispositions before starting any Phase 2 preparation.

## Helpdesk / Enablement Owner

First action: collect known gaps that will affect user expectations, especially DMs, Slack Connect, workflows, and integrations.
